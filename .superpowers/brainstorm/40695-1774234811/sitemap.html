Site Map & Page Structure

All pages are public to view. Actions require Discord login.

Budget Commander League — Site Map

Public Pages

/ — Home

Hero banner, current season info, active budget poll (if polling), recent games, quick leaderboard preview

/leaderboard — Leaderboard

Season standings table: rank, player, wins, losses, win rate. Click a player to see their profile with per-deck stats.

/games — Game History

List of all games: date, 4 players, decks used, winner. Filter by season/player.

/players/[id] — Player Profile

Discord avatar, overall stats, deck collection with per-deck win rates, game history.

/poll — Budget Poll

Vote for $20/$50/$100. Shows live results. Locked once admin closes it.

Authenticated Pages

/dashboard — My Dashboard

Pending game confirmations, my stats, my decks, quick actions.

/decks/new — Register Deck

Form: deck name, commander name, external link (Moxfield/Archidekt).

/games/new — Log a Game

Select 3 other players, each picks a deck, select winner. Creates PENDING game for others to confirm.

/games/[id] — Game Detail

View game details. If participant & pending: confirm or dispute.

Admin Pages

/admin — Admin Panel

Manage seasons (create/end), lock polls, resolve disputed games, manage users (ban/remove), edit game results.
